### Sandwich Maker API:  
* A backend API that simulates a sandwich maker system supporting CRUD operations.
* The application is built using FastAPI and connects to a MySQL database to store data.

* This project was developed to practice backend development concepts such as API design, database integration, and CRUD operations.

* Technologies: Python, FastAPI, MySQL, Git

### Run with:
`uvicorn api.main:app --reload`

* Features:
  - Create, read, update, and delete sandwiches
  - RESTful API design
 
* Author: John (JJ) Botticchio
